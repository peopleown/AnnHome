### 基于wordpress的模板
************************************************************************

#### 简介
+ 本项目基于wordpress，涉及php、html等知识
+ 支持侧边栏小工具、顶部栏工具以及页面模版
+ 本模版适用于绝大数个人博客
+ 本主题不支持ie6

####使用方法
+ 进入你的wordpress后台，点击主题
+ 上传本项目的打包zip文件安装
+ 进行相关主题设置
